# 🧠 Quiz App

A dynamic and user-friendly Quiz Application built using Python, Tkinter, and Object-Oriented Programming principles. This app fetches real-time quiz questions from an online API and provides an interactive interface for users to test their knowledge.

## 📝 Description

This project is a quiz game that presents multiple-choice questions retrieved from an external trivia API. The app uses a graphical interface created with Tkinter and follows a clean OOP structure for modular, maintainable code.

Users answer one question at a time, receive immediate feedback, and get their final score at the end of the quiz.

## ⚙️ Features

- Fetches questions from a trivia API (e.g., Open Trivia DB)
- Multiple-choice questions with real-time interaction
- Instant feedback on whether answers are correct or incorrect
- Score tracking throughout the session
- Simple and responsive GUI using Tkinter
- Built using Object-Oriented Programming for clarity and reusability

## 🧩 Technologies Used

- **Python 3**
- **Tkinter** – for GUI
- **Requests** – for API calls
- **OOP principles** – for organized and scalable code
